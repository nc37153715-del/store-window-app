■ 배포 화면 (브라우저에 이미 열렸을 수 있음)
----------------------------------------------------------------
https://share.streamlit.io/deploy?repository=nc37153715-del/store-window-app&branch=main&mainModule=app.py

1) GitHub 계정으로 Streamlit 로그인 / Authorize
2) App 설정 확인
   - Main file path: app.py
3) Advanced settings → Secrets 에 아래 한 줄 추가:

   OPENAI_API_KEY = "sk-...."

   ※ 바탕화면 매장평가_Window\.env 파일에 있는 키를 복사
4) Deploy!


■ 완료 후
----------------------------------------------------------------
생기는 주소 예: https://store-window-app-xxxxx.streamlit.app
→ 핸드폰 브라우저에 북마크
→ PC가 꺼져 있어도 이 주소로 사용
